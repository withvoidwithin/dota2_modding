# О проекте

Добро пожаловать в **D2_MODDING**!
Этот ресурс задуман как фундаментальная база знаний для всех, кто занимается разработкой пользовательских игр (Custom Games) в Dota 2.

Здесь систематизирована информация о работе с инструментарием Dota 2: архитектура, структура файлов, методы API, Panorama UI, KV-таблицы и другие недокументированные возможности движка.

<Aside type="caution" title="Экспериментальный проект">
	Проект находится на ранней стадии разработки и поддерживается силами **одного автора**.
</Aside>

## Полезные ресурсы

Информации о моддинге в сети немного, и она часто фрагментирована. Помимо этого сайта, рекомендую обращаться к следующим источникам:

- [ModDota](https://moddota.com/api) — основной справочник с дампом API и базовыми гайдами.
- [Valve Developer Community](https://developer.valvesoftware.com/wiki/Dota_2_Workshop_Tools) — официальная Wiki от сообщества.
Обновляется редко и местами содержит устаревшую информацию, но необходима для понимания базовых принципов Source 2.
- [YouTube | CoolStoryBob](https://www.youtube.com/@CoolStoryBob-or6hv/videos) — Канал разработчика с простыми гайдами для начинающих в юмористической подаче.
