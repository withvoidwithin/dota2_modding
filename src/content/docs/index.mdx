# О проекте

Добро пожаловать в проект, посвященный моддингу **Dota 2**!
Этот сайт создан как обширный ресурс для всех, кто занимается созданием собственных пользовательских игр.

Здесь **будет** собрана различная информация о работе с функционалом инструментов: систем, редакторов, API и прочих особенностях всего SDK.

<Aside type="caution" title="Эксперементальный проект">
	Проект находится на начальной стадии разработки и поддерживается силами **одного автора**.
</Aside>

## Полезные ресурсы

В интернете разбросано полно полезной информации связанной с моддингом Dota 2. Вот несколько рекомендуемых ресурсов:

- [ModDota](https://moddota.com/api) — очень полезный ресурс с дампом API игры и гайдами.
- [Valve Developer Community](https://developer.valvesoftware.com/wiki/Dota_2_Workshop_Tools) - wiki от сообщества модмейкеров.
Крайне редко обновляется, но позволяет понять фундаментальную базу. Некоторая информация может быть **не актуальной или устаревшей**.

<Aside type="danger" title="Дисклеймер">
	Материалы, представленные на этом ресурсе, являются результатом эмпирического исследования.
	Движок Source 2 — это закрытая экосистема, и вся информация здесь — это моя личная субъективная интерпретация работы внутренних механик API, полученная за годы практической разработки.
</Aside>
